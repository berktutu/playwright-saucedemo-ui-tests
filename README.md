# SauceDemo Playwright UI Automation Tests

## The goal of the project

This project demonstrates fully automated testing of an application without access to its internal structure (black-box testing). The objectives are to:

- Preparing testing documentation.
- Test in the scope of the project.
- Identifying and reporting UX/UI issues.
- Using Playwright to execute the test cases

## Live Server

https://www.saucedemo.com

## Test Scope

- Login
- Item Catalog
- Cart Functionality
- Checkout
- UX/UI

## Tech Stack

- Node.js
- Typescript
- Playwright
